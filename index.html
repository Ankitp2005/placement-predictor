How this was built

Three datasets in, before we trusted any of it.

Worth saying plainly — the first two didn't hold up, and we're only showing this one because it did.

Ruled out a false start

A 100,000-row dataset let a model memorise its training data completely, yet it still couldn't beat 0.55 AUC on unseen students. No real signal — so we didn't ship it.

Found signal, wrong scale

A 215-student dataset showed strong signal (0.94 AUC) but came from one college, one cohort. Too narrow to trust for students outside it.

Landed on scale and signal

10,000 engineering students, correlations up to 0.75. Tested four algorithms under cross-validation before picking LightGBM — it tied for best and trains fastest.

Before you rely on this

What this model is, and isn't.

This estimates odds from patterns in one training set — it doesn't know your college's recruiters or this year's hiring market. Treat a low estimate as a prompt to work on backlogs and skills first, not as a verdict. A high estimate is reason for cautious optimism, not complacency.
